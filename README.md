
# Week 1 Operations Analyzer

## Project Overview

This project automates operational efficiency tracking across hospital wards. The notebook calculates efficiency percentages, classifies operational status levels, and generates reports to identify areas needing attention.

## Files Included

### week1_ops_analyzer.ipynb

Contains:

* KPI efficiency calculation
* Operational status classification
* Data modeling for three hospital wards
* Automated reporting process
* Documentation written for junior analysts

### Week1_Memo_EdnahKimani.pdf

A business memo explaining:

* Why efficiency tracking matters
* How automation improves operational visibility
* Recommendations for deployment
